# lature

## Brief

`lature` stands for `lua nature`.

In this project, I use `LOVE2d` to simulate the `natural system`.

## Usage

- **WSL**

You need to update the `LOVE2d` path in `build.lua`

```console
$ lua build.lua chapter0/random_walker
```

- **OTHER**

```console
$ love.exe chapter0/random_walker 

OR

$ love chapter0/random_walker 
```

## Reference

- Book: `The Nature of Code Simulating Natural Systems with Processing (Daniel Shiffman)`
